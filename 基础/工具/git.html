    对象和内存的寻址

    git中的对象可以是blob， 树，或提交

    type object = blob | tree | commit
    1

    git在存数据的时候，所有的对象会基于他们的SHA-1哈希进行寻址

    objects = map<string,object>;
def store(object):
		// sha1哈希算法
  	id = sha1(object)
		objects[id] = object
def load(id):
	return objects[id]
